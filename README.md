# csv_plotter.py

Interactively plot CSV files.

## Features
- matplotlib's plot window features:
    * image saving
    * mouse interaction (zoom, crop...)
- select which columns to plot
- select X axis

## Non-features
- CSV edition (use a text/spreadsheet editor)
- open more than one file
